# DealHub Features

## Core Features

###  Implemented

1. **Card-First Browsing (Map View Coming Soon)**
   - Focused, photo-forward cards optimized for discovery
   - Airbnb-inspired design aesthetic
   - Responsive layout for all devices

2. **Category Filtering**
   - All, Restaurant, Grocery, Gas, Coffee
   - Visual category icons
   - Smooth animations

3. **Smart Sorting**
   - Closest to Me (based on geolocation)
   - Highest Discount
   - Trending (based on views/clicks)

4. **Photo-Focused Deal Cards**
   - Large, high-quality images
   - Discount badges prominently displayed
   - Multiple badges (Great Deal, Ends Soon, Trending, New)

5. **Multiple Deals Per Store**
   - Horizontal carousel on cards
   - Swipe/scroll to see additional deals
   - Visual indicators for multiple deals

6. **External App Integration**
   - Click deals to open partner apps
   - Seamless redirect to company apps
   - Partner app branding

7. **SEO Optimization**
   - Meta tags and descriptions
   - Structured data (JSON-LD)
   - Sitemap generation
   - Robots.txt configuration
   - Open Graph tags

8. **Monetization Hooks**
   - Click tracking
   - View tracking
   - Conversion tracking
   - Affiliate rate management
   - Sponsored deal detection

## Technical Stack

- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Maps**: Google Maps API (optional, currently disabled)
- **SEO**: Next.js built-in SEO features
